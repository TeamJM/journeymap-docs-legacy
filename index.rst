Home
====

JourneyMap is a popular mapping mod for Minecraft_. Originally released
by techbrew in 2011, it's gone from strength to strength and today
stands as the most popular mapping mod in the world.

.. figure:: /_static/images/background.jpg
    :alt: Sample map created with JourneyMap

JourneyMap includes an in-game minimap and full-screen map, as well
as a webmap that can be viewed in a browser.

If you'd like a featureful, easy-to-use mapping mod then why not give
JourneyMap a try? You can `find it on Curse`_ and install it just like
any other Forge_ mod - or continue reading if you need help.

.. _find it on Curse: https://www.curseforge.com/minecraft/mc-mods/journeymap
.. _Forge: https://forums.minecraftforge.net/
.. _Minecraft: http://minecraft.net/
